# 운영체제란?
- 하드웨어를 관리하고, 컴퓨터 시스템의 자원들을 효율적으로 관리하며, 응용 프로그램과 하드웨어간의 인터페이스로 다른 프로그램이 유용한 작업을 할 수 있도록 환경을 제공해준다.
- ex) Windows, UNIX, MS-DOS
---
## 운영체제의 역할
```
프로세스 관리, 저장장치 관리, 네트워킹, 사용자 관리, 디바이스 드라이버
```
#### 1. 프로세스 관리
운영체제에서 작동하는 프로그램을 관리하는 기능이다.  
프로세서를 관리하는 것이라고도 볼 수 있다. 현재 CPU가 어떤 행동을 해야할 지 정의한다.  
현재 CPU가 어떤 프로세스를 점유해야 하는지, 실제로 CPU에 할당하고 프로세스 간의 공유 자원 접근을 관리한다.
#### 2. 저장 장치 관리  
1차 저장장치(메모리), 2차 저장 장치(하드디스크) 등을 관리하는 기능이다.
- 1차 저장장치(Main Memory)
  - 프로세스에 할당하는 메모리 영역의 할당과 해제
  - 각 메모리 영역 간의 침범 방지
  - 메인 메모리의 효율적 활용을 위한 가상 메모리 기능
- 2차 저장장치(HDD, NAND Flash Memory 등)
  - 파일 형식의 데이터 저장
  - 파일 데이터 관리를 위한 파일 시스템을 OS에서 관리
  - 현재 많은 파일 시스템이 개발되어서 사용되고 있음
#### 3. 네트워킹
TCP/IP 기반의 인터넷에 연결하거나, 응용 프로그램이 네트워크를 사용하려면 운영체제에서 네트워크 프로토콜을 지원해야한다.  
현재 상용 OS들은 다양하고 많은 네트워크 프로토콜을 지원한다.
운영체제는 사용자와 컴퓨터 하드웨어 사이에 위치하여, 하드웨어를 운영 및 관리하고 명령어를 제어하여 응용 프로그램 및 하드웨어를 소프트웨어적으로 제어 및 관리를 해야한다.
#### 4. 사용자 관리
PC은 단 한명의 것이 아니다.  
하나의 PC로 여러 사람이 사용하는 경우가 있다. 그렇기 때문에 운영체제는 각 계정을 관리하는 기능이 있어야 한다.  
사용자 별로 보안을 강화하여 개인 파일에는 다른 사용자가 접근하지 못하도록 해야한다.  
이 밖에도 파일이나 시스템 자원에 접근 권한을 지정할 수 있도록 지원하는 것이 사용자 관리 기능이다.
#### 5. 디바이스 드라이버
운영체제는 여러 디바이스를 관리한다. 키보드, 프린터, 모니터 등등.. 이런 하드웨어와 상호 작용 하기위해서는 별도의 드라이버 설치가 필요하다.  
운영체제에서는 이러한 외부 기기들과 서로 약속하여 추상화 해주는 계층이 있고 이 계층을 디바이스 드라이버 라고 부른다.  
하드웨어가 굉장히 많이 존재하기 때문에 운영체제 내부의 디바이스 드라이버도 많이 존재한다.  
이러한 기능을 담당하는 것도 운영체제의 역할이다.

### Q1. 프로세스와 스레드의 가장 큰 차이는 무엇인가요?
메모리 공간의 독립성이 가장 큰 차이 입니다.  
프로세스는 독립적인 공간을 가집니다 -> 코드 데이터 힙 스택  
프로세스와 프로세스는 메모리가 격리되어 있습니다. 스레드는 같은 프로세스 내부에서 메모리를 공유합니다.  
코드, 데이터, 힙은 공유하고 스택만 독립적입니다. 스레드는 프로세스 없이 독립적으로 존재할 수 없습니다.  
### Q2. 프로세스 관리에서 운영체제가 담당하는 핵심 기능 두 가지를 말해보세요.
운영체제에서 작동하는 프로그램을 관리하는 기능이다. 프로그램을 프로세스로 생성하고, 프로세스들의 상태를 관리한다.  
그리고 현재 CPU가 어떤 프로세스를 점유해야하는지 CPU를 제어하는 기능을 수행한다.
### Q3. 1차 저장장치(Main Memory)를 운영체제가 관리할 때 수행하는 기능은 무엇인가요?
프로세스를 메모리 영역에 할당시키고 해제하는 작업을 한다. 각 메모리 영역간의 침범을 방지하도록 조절한다.  
### Q4. 2차 저장장치를 관리하기 위한 운영체제 구성 요소는 무엇인가요?
HDD, 외부 저장소에 파일 형식으로 데이터를 저장한다. 파일 데이터 관리를 위한 파일 시스템을 OS에서 지원한다.
### Q5. 운영체제가 네트워킹 역할을 하기 위해 반드시 제공해야 하는 것은 무엇인가요?
응용 프로그램이 네트워크를 사용하기 위해서는 운영체제에서 네트워크 프로토콜을 지원해야한다.  
현재 대부분의 OS에서는 네트워크 프로토콜을 지원한다.  
OS는 패킷 전송, 주소 지정, 오류 검출 등 기본 네트워크 기능을 제공하고, 하드웨어에 직접 접근하지 않고 통신할 수 있는 인터페이스를 제공한다.  
### Q6. 사용자 관리 기능이 필요한 이유를 한 문장으로 설명해보세요.
하나의 PC를 한 명의 사용자가 사용하리라는 법은 존재하지 않습니다. 이러한 이유 때문에 사용자 관리 기능이 필요합니다.  
사용자 관리 기능을 통해 서로 다른 사용자가 각자의 파일이나 정보에 침입하지 못하게 하여 보안을 강화시킵니다.
### Q7. 디바이스 드라이버가 존재해야 하는 이유는 무엇인가요?
디바이스 드라이버가 존재하지 않으면 외부 기기를 사용할 수 없기 때문입니다. 외부 기기, 키보드, 마우스 등등의 기기를 사용하기 위해서는 OS에서 추상화된 인터페이스를 통해 내부 소프트웨어와 하드웨어가 상호작용합니다.  
이를 중간에서 관리하는 매개체가 OS이고, 이 드라이버가 곧 추상화된 인터페이스입니다.
